POS Order Reprint

Salient Features
  • Seller can re-print previous orders in the running session.
  • Seller can see/re-print all orders after a specific date configured in backend.
  • Seller can also see/re-print orders associated with individual customers.
  • Seller can re-print receipt in normal pdf format or in ticket format(Order receipt).
  • Seller can also re-print receipt by using POSBOX.
  • POS Session View
  • Backend Configuration

Backend View And Configuration

These are the configuration settings which specify how many orders from the past would be loaded in POS when it starts.

Select between option to print duplicate receipt in given format
  • Select 'Browser Based' to print/download duplicate receipt in pdf format.
  • Select 'POSBOX' to print receipt as xml receipt(print receipt by using POSBOX).
  • Select 'POS Ticket' to print duplicate receipt as order receipt.
